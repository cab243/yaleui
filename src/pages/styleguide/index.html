About the Yale UI Style Guide

The Yale UI Style Guide was made to help developers build interfaces that match Yale's visual identity, and follow WCAG 2.0 accessibility recommendations. The style guide relies on:

  • Yale's Identity Guidelines for the web
  • Usability & Web Accessibility

Help & Support

The Yale UI Style Guide was built with Foundation for Sites. For more information on how to use this responsive front-end framework, check out the documentation or get help from the Foundation community.
